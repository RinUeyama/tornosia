# スキル

PCは 4+9+4 種類のスキルを持つ。ボスは各々に固有。

## パッシブスキル

- 常時発動
- PCボスともにお互いに公開
- 1つ目の通常スキルのあとに1つ選択

## 通常スキル

- ゲーム開始時に3つ選択
- MPを使用して各ターン1つまで使用可能

## Ultスキル

- ゲーム開始時にランダムな発動条件が設定される
- 発動条件を満たしたとき、2つから1つを選択
- 原則として戦闘中1度しか使えない

### 発動条件一覧

一度でも満たせばよい

- 自身のHPが60％以下
- ボスのHPが50％以下
- だれかのHPが30％以下（これはボスも含む）

## スキルの構成

- 攻撃力：使用時に対象へダメージを与える
- 防御力：自身にそのターンのみ有効な防御力を与える
- 対象：自身／自身の周囲／味方全体／相手／相手後衛／相手前衛／相手全体
- 副作用：使用時に発生する副次効果
- コスト：MP消費量
